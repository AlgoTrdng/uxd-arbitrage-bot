# UXD arbitrage bot

## Setup

- Install dependencies

```sh
pnpm i
```

- Set env variables in `.env`

```env
SOL_RPC_ENDPOINT=solana RPC endpoint url
SOL_PRIVATE_KEY=solana private key byte array as string
```

## Start bot

- Development
```sh
npm run start:dev
```

- Production
```sh
npm run start:prod
```
